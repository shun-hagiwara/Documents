<h1>SQLと連動したWEB掲示板</h1>

>"名前"と"コメント"を打ち込むことができる掲示板です。
>この掲示板には、"投稿機能""削除機能""編集機能"の3つの機能があります。

<p align="left">
  <img width="400" src=https://github.com/shun-hagiwara/Documents/assets/147070848/a176b57d-1943-4b78-b5d3-7b063145fd65" />
</p>

<br>
<h2>投稿機能："名前""コメント"を投稿することができます。</h2>

```
投稿前
```

<p align="left">
    <img width="400" src="https://github.com/shun-hagiwara/Documents/assets/147070848/bd3688e7-12ed-4b9a-84ae-4a00865a6891"/>
</p> 

```
投稿後
```
>名前、コメント、パスワードを送信すると、　最新投稿の次の投稿として追加されます。

<p align="left">
<img width="400"  src="https://github.com/shun-hagiwara/Documents/assets/147070848/338ec68d-51e7-4d52-a6c1-24627c5b6631](https://github.com/shun-hagiwara/Documents/assets/147070848/338ec68d-51e7-4d52-a6c1-24627c5b6631"/>

<h2>削除機能：特定の投稿番号を指定し、"名前""コメント"を削除できます。</h2>

```
削除前
```
>今回の場合は投稿番号"1"を削除するために、投稿した際に設定したパスワードである"1"を入力します。
<img width="400" alt="スクリーンショット 2023-10-08 18 51 03" src="https://github.com/shun-hagiwara/Documents/assets/147070848/9938e894-9df9-4d8a-ae3d-bff07ef7ed9f">

```
削除後
```
>パスワードが正しければ、指定した投稿番号の投稿が削除されます。
<img width="400" alt="スクリーンショット 2023-10-08 18 51 14" src="https://github.com/shun-hagiwara/Documents/assets/147070848/58e08f88-fa97-4e71-9a2b-079231eede52">

<h2>編集機能：特定の投稿番号を指定し、"名前""コメント"を編集できます。</h2>

```
編集フォーム送信前
```
>今回の場合は投稿番号"5"を編集するために、投稿した際に設定したパスワードである"1"を入力します。
<img width="400" alt="スクリーンショット 2023-10-08 18 51 51" src="https://github.com/shun-hagiwara/Documents/assets/147070848/864c7212-41f4-4806-8ae2-a6414d2bc69c">

```
編集フォーム送信後
```
>パスワードが正しければ、指定した投稿番号の"名前""コメント""パスワード"が入力フォームに表示されます。
<img width="400" alt="スクリーンショット 2023-10-08 18 52 26" src="https://github.com/shun-hagiwara/Documents/assets/147070848/d650c0fb-3ca7-47d0-94d9-e00613171b86">

```
入力フォーム送信前
```
>今回は"名前"は変更せず、"コメント"を"午前中は休みです"に書き換えます。
<img width="400" alt="スクリーンショット 2023-10-08 18 52 52" src="https://github.com/shun-hagiwara/Documents/assets/147070848/2c1ebed5-83b0-4308-ba4f-073709419c78">


```
入力フォーム送信後
```
>送信後、編集が完了します。
